# Institutional Website – Dra. Luciana Pereira

This project consists of the development of an institutional website for promoting the professional services of physiotherapist Dra. Luciana Pereira.

The main goal of the website is to strengthen the professional’s online presence, make it easier for patients to get in touch, and present her services in a clear and professional way.

##  Features

- Responsive layout (desktop, tablet and mobile)
- Professional introduction section
- Services overview
- Working hours and availability
- WhatsApp integration
- Social media links
- Mobile-friendly navigation (hamburger menu)

##  Technologies used

- HTML5
- CSS3
- JavaScript
- Deployment via Netlify

##  Live version

The website is hosted on Netlify and available at:

 https://dralucianafisio.netlify.app/  
👉(custom domain currently being configured)

##  Notes

This project was developed as part of academic and practical experience within the Systems Analysis and Development course.

It is also included in the developer’s professional portfolio.

# Site Institucional – Dra. Luciana Pereira

Este projeto consiste no desenvolvimento de um site institucional para divulgação dos serviços profissionais da fisioterapeuta Dra. Luciana Pereira.

O objetivo principal do site é fortalecer a presença digital da profissional, facilitar o contato com pacientes e apresentar seus serviços de forma clara, acessível e profissional.

## 🧩 Funcionalidades

- Layout responsivo (desktop, tablet e mobile)
- Seção de apresentação profissional
- Lista de serviços oferecidos
- Informações de atendimento
- Integração com WhatsApp
- Links para redes sociais
- Navegação otimizada com menu mobile (hamburger)

##  Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Deploy via Netlify

##  Publicação

O site está hospedado na plataforma Netlify e disponível em:

 https://dralucianafisio.netlify.app/  
 (domínio personalizado em configuração)
